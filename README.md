## Eksempel av MSW For Bekks julekalender 2024
Her finner du et veldig enkelt eksempel på hvordan sette opp MSW for en React-applikasjon.

1. Klon repoet
2. Kjør `npm install`
3. Kjør `npm run dev`
4. Åpne nettleseren, endre på mock-dataene i handlers.ts og se at dataene endrer seg.

For mer info besøk sidene til [https://mswjs.io/](https://mswjs.io/)
